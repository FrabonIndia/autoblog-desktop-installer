## ‚ùì **TROUBLESHOOTING:**

### **App Won't Open:**
- **Windows:** Right-click installer ‚Üí "Run as Administrator"
- **Mac:** Right-click app ‚Üí "Open" (bypasses security)

### **Can't Generate Posts:**
- Check your OpenAI API key is correct in Settings
- Make sure you have billing set up with OpenAI
- Check your internet connection

### **WordPress Publishing Not Working:**
- Verify your WordPress URL is correct (include https://)
- Make sure you're using an Application Password (not your regular password)
- Check that WordPress REST API is enabled
